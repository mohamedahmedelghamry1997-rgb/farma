# **App Name**: STUDIO FIREBASS AI

## Core Features:

- User Authentication & Role Management: Secure sign-in and management for Client, Broker, Supervisor, and Admin roles, controlling access to app functionalities based on assigned permissions.
- Interactive Chalet & Booking Display: Clients can elegantly browse chalets via card-based design, view main images, prices, and mock ratings, and select available dates from a dynamic calendar enforcing back-to-back bookings.
- Booking Submission & Workflow: Clients submit booking requests with personal details and special notes. The system facilitates a workflow for Admins to confirm or cancel these bookings, visible to Brokers and Supervisors as needed.
- Chalet Content & Pricing Management: Brokers and Admins can update chalet details including descriptions, image galleries, and pricing. Admins have full control over all chalet settings and booking rules.
- Check-in/Check-out & Condition Reporting: Supervisors use a dedicated interface to record client check-in and check-out times, document chalet condition with notes (e.g., maintenance needed, cleanliness), and add security deposit information.
- Comprehensive Admin Dashboard & Alerts: A full administrative panel allows Admins to manage all users, review/confirm/cancel bookings, track client details and booking specifics, and receive alerts for conflicts or supervisor reports.
- AI-Powered Booking Gap Assistant: An AI tool for Admins that analyzes the chalet booking calendar to proactively identify potential empty days and suggests optimal booking configurations or strategies to maintain the required continuous, back-to-back occupancy.

## Style Guidelines:

- The visual scheme embraces a calm and modern luxury. A primary color inspired by the serene sea, #247ABE (a deep, calming blue), serves as the dominant interactive and branding hue. The background color, #ECF5FA, is a very light, subtle off-white with a hint of cool blue, ensuring a clean and spacious feel. Accent elements, like buttons or highlights, utilize a sophisticated golden touch, #E6A926, to convey elegance without overpowering.
- The font of choice is 'Inter', a modern grotesque sans-serif. It is used consistently for both headlines and body text to ensure a clean, objective, and highly legible experience, reflecting a contemporary and polished aesthetic suitable for a global and professional application.
- Clean, minimalist line-art icons should be used throughout the application to maintain a sleek, modern, and uncluttered visual style consistent with the card-based design and luxury feel. Icons should be easily understandable across all user roles.
- A 'mobile-first' and 'card-based' design approach creates an experience akin to native Android/iOS applications. Each chalet, booking, or actionable item is presented in elegant, easily digestible cards. The layout prioritizes responsive adaptability, ensuring seamless interaction across all device sizes with an emphasis on clarity and efficient navigation for all user roles.
- Subtle and tasteful animations are integrated to enhance user experience. These include smooth transitions between pages, gentle hover effects on interactive elements, and brief loading indicators. Animations are designed to feel fluid and contribute to the app's premium, responsive feel without being distracting.
